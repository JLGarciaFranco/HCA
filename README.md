# Herramientas Computacionales Avanzadas para Ciencias de la Tierra

Repositorio del curso **Herramientas Computacionales Avanzadas**, impartido por  
**M. Claudia Peña Mata** y **Dr. Jorge Luis García Franco**.

Este curso introduce el uso de herramientas modernas de programación científica para el análisis, modelación y visualización de datos en Ciencias de la Tierra. A lo largo del semestre se desarrollan habilidades en pensamiento computacional, análisis numérico, visualización científica y procesamiento de datos geoespaciales, utilizando Python como lenguaje principal.

---

## Unidad 1 — Introducción al Análisis Computacional en Ciencias de la Tierra
Se establecen los fundamentos del pensamiento computacional y la programación científica aplicada al análisis de datos geocientíficos. Se revisan estructuras básicas de programación, pseudocódigo y algoritmos, así como los conceptos esenciales del lenguaje Python.

Temas principales:
- Variables, tipos de datos y operadores
- Condicionales y bucles
- Funciones y estructuras de datos
- Introducción a librerías esenciales: **NumPy** y **Pandas**

---

## Unidad 2 — Visualización de Datos
Se introduce la importancia de la visualización científica para el análisis exploratorio de datos en Ciencias de la Tierra. Se revisan formatos de datos comunes (CSV, NetCDF, HDF5) y se desarrollan habilidades para la creación de gráficos científicos claros y reproducibles.

Temas principales:
- Manejo de bases de datos científicas
- Visualización con **Matplotlib**
- Análisis exploratorio con **Seaborn**
- Personalización y exportación de figuras

---

## Unidad 3 — Métodos Numéricos Aplicados
Se presentan métodos numéricos fundamentales utilizados en modelación ambiental y geofísica, incluyendo técnicas de interpolación, ajustes de datos y simulación de procesos físicos mediante modelos discretos.

Temas principales:
- Interpolación y ajuste de datos
- Solución numérica de ecuaciones diferenciales ordinarias
- Modelos de transporte y difusión de contaminantes

---

## Unidad 4 — Análisis Geoespacial de Datos
Se introduce el análisis espacial aplicado a Ciencias de la Tierra mediante el uso de librerías modernas para el manejo de datos vectoriales y raster. Se desarrollan habilidades para la generación de mapas temáticos y el análisis de datos georreferenciados.

Temas principales:
- Manipulación de datos espaciales con **GeoPandas**, **Cartopy** y **Rasterio**
- Análisis raster y vectorial
- Generación de mapas temáticos
- Series temporales georreferenciadas

---

Este repositorio contiene notebooks, datos de ejemplo y materiales de clase diseñados para apoyar el aprendizaje práctico y reproducible del análisis computacional en Ciencias de la Tierra.
